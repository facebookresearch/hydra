Hydra

Flexible experimentation framework

Get Started

Easy to Use

Hydra was designed from the ground up to be easy to use and allow you to skip the boilerplate code and jump straight into the code you are actually interested in writing.

Focus on What Matters

Hydra lets you focus on your Experiment, and reduces the boilerplate code you need to write. For example:

  • Command line flags
  • Loading configuration files
  • Output directory management
  • Logging configuration
  • Launching to a cluster

    • Plugable architecture

    Hydra has a Pluggable architecture, allowing it to be extended to integrate with your own infrastructure while maintaining the rest of the flexibility and power it provides.
