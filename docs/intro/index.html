Introduction

Hydra is a Python library that help you write flexible command line applications, without all the boilerplate code.

Hydra provides a unified command line interface to run your application locally or remotely. it can compose the configuration for your application dynamically, while allowing free overriding of everything in the config from the command line.

The best way to understand what Hydra is good for is to go through some examples.
